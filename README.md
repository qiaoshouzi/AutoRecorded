## 目录解释

| 文件名     | 解释                                    |
| ---------- | --------------------------------------- |
| /doc       | 帮助文档                                |
| /ffmpeg    | 历史ffmpeg的.zip文件                    |
| /log       | 存储log文件的文件夹（运行时会自动生成） |
| /old       | 早期开发的文件（过于憨                  |
| ffmpeg.exe | ffmpeg最新构建版，一般不动              |
| start.py   | 运行文件                                |
